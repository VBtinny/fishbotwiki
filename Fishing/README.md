# How?

To fish, type `!fish`

To be more specific on your desired length of fishing, you can declare a length

example: '!fish xl`

all lengths: `xs`,`s`,`m`,`l`,`xl`,`xxl`

# Fishing Lengths

A new player can:
  - s
  - m
  - l
